## 🚀 실제 서비스로 전환하기

### **필요한 경우:**

✅ 실제 영수증 내용을 자동으로 추출하고 싶을 때  
✅ 30명 직원이 실제 운영 환경에서 사용할 때

### **10분 만에 전환:**

#### **1단계: Google Cloud API 키 발급 (10분)**

```bash
1. https://console.cloud.google.com 접속
2. 프로젝트 생성: "하이콘-영수증-스캔"
3. Cloud Vision API 활성화
4. API 키 생성 → 복사 (AIzaSy로 시작하는 39자 문자열)
```

#### **2단계: Supabase에 등록 (1분)**

```bash
1. https://supabase.com/dashboard 접속
2. Project Settings → Edge Functions → Secrets
3. GOOGLE_VISION_API_KEY 찾기
4. 편집 → 새 API 키 붙여넣기 → Save
```

#### **3단계: 작동 확인 (1분)**

```bash
1. 직원 관리 → 직원 지출 내역
2. 📸 영수증 스캔
3. 🔍 Google Vision API 연결 상태 확인
   → ✅ "정상적으로 설정되었습니다!"
4. 실제 영수증 촬영 및 업로드
   → ✅ "영수증 자동 인식 완료! (Vision AI)"
```

**✅ 완료! 실제 서비스 시작!**
